[상태 관리 모듈]

이 모듈은 상태 머신 프레임워크 내에서 상태를 효율적으로 관리하기 위해 설계되었습니다. 테스트를 위해 임시로 UI 매니저를 구현하여 현재 상태에 따라 특정 UI 요소를 열도록 설정했습니다.

구조 및 구현

IState 인터페이스:

IState 인터페이스는 각 상태가 구현해야 할 메서드들을 정의합니다.
StateMachine 추상 클래스:

StateMachine 클래스는 IState 인터페이스를 상속받아 상태 진입 및 종료와 같은 공통 동작을 포함하는 추상 클래스를 만듭니다.
구체적인 상태 클래스:

각 동작에 사용될 상태들은 StateMachine을 상속받아 구체적인 클래스로 정의됩니다. 이러한 클래스들은 상태 진입 및 상태 종료 시 발생할 상황을 구체적으로 구현합니다.
스크립터블 오브젝트 사용:

상태를 동적으로 관리하기 위해 스크립터블 오브젝트를 사용하여 데이터 리스트에 있는 상태 프리팹들을 생성하고 등록함으로써 유연성을 높였습니다.
이 구조를 통해 상태 관리가 유연하고 확장 가능하게 되어 다양한 상황에 맞게 쉽게 조정할 수 있습니다. 각 상태 클래스는 구체적인 동작을 구현하며, 상태 전환 시 필요한 초기화 및 정리 작업을 명확하게 정의합니다.
